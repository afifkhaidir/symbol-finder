## Installation

### Manual Download

- [Download](../../releases/latest/download/symbol-finder.sketchplugin.zip) the latest release of the plugin
- Un-zip
- Double-click on symbol-finder.sketchplugin

### Using Sketch Runner

- Open `plugins > Runner > install`
- Type `symbol-finder`
